# Order-Sales Dashboard

## Description
A data visualization dashboard analyzing order and sales performance to track revenue, order trends, and business insights.

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Key Insights
- Highest sales from Maharashtra
- Clothing category has highest quantity
- Profit varies across months
